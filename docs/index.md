# JustDoIt

Zero-dependency Python 3 ASCII art CLI. Renders text as ANSI-colorized terminal output using bitmap fonts, FIGlet fonts, TTF rasterization, and a composable effects pipeline.

- [Static Gallery](https://hogjonny-amzn.github.io/JustDoIt/gallery/) — fonts, fills, color, spatial & 3D
- [Wide Gallery](https://hogjonny-amzn.github.io/JustDoIt/gallery-wide/) — HiDPI / retina renders
- [4K Gallery](https://hogjonny-amzn.github.io/JustDoIt/gallery-4k/) — full 3840×2160 renders
- [Animation Gallery](https://hogjonny-amzn.github.io/JustDoIt/anim_gallery/) — APNG + asciinema casts
- [GitHub](https://github.com/HogJonny-AMZN/JustDoIt)

## Quick Start

```bash
uv sync --dev
python justdoit.py "Just Do It" --color rainbow
python justdoit.py "HEY" --font big
python scripts/demo.py
```
